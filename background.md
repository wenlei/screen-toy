# 项目背景

## 触点：人与 Agent 的关系

我们的作品关注一个核心问题——**人如何与 Agent 相处**。

大语言模型驱动的 Agent 强大、完善、知识广博，但它不"亲人"。繁复的配置、skill 的反复调配、冰冷的一问一答——这些都让人感到距离。技术上令人惊叹，情感上却拒人千里。

## 想法：一个同路人

现代年轻人有热情、有活力、敢想敢做，也同样有压力、有困惑。他们需要同路人，需要心理按摩——不是一个无所不知的答案机器，而是一个愿意陪在身边、偶尔说句俏皮话的伙伴。

于是我们以刘看山的形象，做了一款 **Screen Toy**。它蹲在桌面上，不是工具，是同伴。

## 方式

### 形象与风格

我们参考了加菲猫、xkcd、史努比以及海鸥漫画——顽皮、可爱、亲人，又带一点若有所感的独白。通过动画和文字气泡，我们在代码里埋进对生活的共鸣。我们相信，屏幕外一定有很多想法一样的人，看到这只狐狸会会心一笑。

### 知乎 Agent 连接

作为为知乎平台构建的应用，Screen Toy 接入了知乎的 API，连接知乎庞大的知识语料库。但不止于此——为了给 Agent 增加"人味儿"，我们借助 MBTI 的四维性格组合，构造了 16 种人格作为 System Prompt，让回答带上个性，消减文本阅读的"书卷气"。

### 动画系统

参考 Google Doodle 带来的文化尊重与情感共鸣，刘看山有三类动画：

- **入场动画**: 启动时播放。针对节日制作了不同的节日形象（如 6 月儿童节、端午节），通过日期自动匹配激活（不暴露在图形界面，在配置文件中进行日期与动画的绑定）。
- **随机动画**: 打喷嚏、融化、戴花环、呼啦圈等，由定时器和菜单触发。
- **退场动画**:应用退出时播放，支持节日覆盖动。

感到打扰时，可以将刘看山拖到屏幕四角——它会安静入睡。双击唤醒，或通过状态栏图标控制显示/隐藏。

### 驻留合理性

为了让一个桌面宠物"有资格"占着屏幕，我们加入了快速菜单：一键打开计算器、记事本、邮件等常用软件，少点几个按钮。还有一个夏天的躲太阳小游戏，增加用户的共鸣感。

## 局限

目前只打包了 macOS 端（`.dmg`），尚未覆盖 Windows、Linux 及 HarmonyOS 用户。
